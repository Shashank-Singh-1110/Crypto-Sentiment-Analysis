Methodology
Research pipeline architecture and tools
8-Module Pipeline
1
Reddit Collection
Arctic Shift API — 11 subreddits, 6 coins + MARKET. Checkpoint/resume, CSV export, gap detection.
2
Price + Macro Data
Binance REST API for OHLCV. yfinance for Oil, DXY, Gold, VIX, S&P 500. Alternative.me for Fear & Greed Index.
3
NLP Preprocessing
spaCy (parser disabled), 38 crypto slang terms, POS tagging, NER, TF-IDF per subreddit/coin.
4
Sentiment Analysis
VADER (52 crypto terms) + FinBERT (ProsusAI/finbert). Composite = 0.7×FinBERT + 0.3×VADER. Engagement weighting + z-score correction.
5
Advanced NLP
BERTopic for topic discovery. 7-class emotion detection. Enhanced NER with crypto entity dictionaries.
6
Correlation Engine
Lagged Pearson (0-3 days), Granger causality, binary classification, ablation study across 5 variants.
7
Dashboard
Flask backend + vanilla JS frontend. Chart.js visualizations. Real-time API calls to SQLite.
8
AI Insight Engine
Llama 3 via Ollama. Context-aware analysis using live sentiment, price, and emotion data from the database.
Tech Stack

Python · Flask · spaCy · HuggingFace Transformers · FinBERT · VADER · BERTopic · scikit-learn · SQLite · scipy · statsmodels · Llama 3 · Ollama · Chart.js · Vanilla JS
